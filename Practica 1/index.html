Tipos de Layouts

Que es un layout?

Un layout en diseño web es la estructura o distribución de los elementos en una página web. Define cómo se organizan las secciones como el encabezado, el contenido principal, la barra lateral, el pie de página y otros componentes dentro de la pantalla.
